Crypto Price Analysis & Forecasting

Description:  
Analyze and predict cryptocurrency prices using historical data. Implements ARIMA, Prophet, and LSTM models 
with an interactive Streamlit dashboard to visualize trends, forecasts, and volatility.


Features
- Fetches historical crypto data from Yahoo Finance.  
- Interactive price and forecast charts.  
- Volatility analysis.  
- Easy-to-use GUI with Streamlit.


Usage
1. Install dependencies:
pip install -r requirements.txt
